Project Title

Real Estate Sales Analysis using Excel, Power Query, and Power BI

Objectives

The primary objective of this project is to analyze real estate property listing data and develop an interactive dashboard that provides meaningful business insights. The specific objectives are:

To clean and prepare raw real estate data using Microsoft Excel.
To transform and validate data using Power Query.
To create meaningful DAX measures in Power BI.
To design an interactive dashboard using KPI cards, charts, and slicers.
To analyze property listings by city, property type, listing status, and price.
To support business decision-making through effective data visualization.

Dataset Information

Dataset Name: Real Estate Property Listings Dataset

Dataset Description:

The dataset contains information related to property listings, including property details, pricing, location, listing status, and agent information.

Key Findings (with Metrics)

The Power BI dashboard provides the following key performance metrics:

Total Revenue: Displays the total value of all listed properties.
Total Properties: Shows the total number of property listings.
Average Property Price: Calculates the average listing price.
Maximum Property Price: Identifies the highest-priced property.
Minimum Property Price: Identifies the lowest-priced property.
Total Cities: Shows the number of unique cities in the dataset.
Total Agents: Displays the number of unique agents managing the properties.

Business Insights
Identified the cities with the highest number of property listings.
Compared average property prices across different property types.
Analyzed monthly property listing trends.
Examined the distribution of listing statuses.
Evaluated the overall performance of property listings using interactive KPIs and charts.
