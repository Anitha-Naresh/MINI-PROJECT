Project Title
Real Estate Sales Analysis using Excel, Power Query, and Power BI

Project Overview
This mini project demonstrates the complete data analytics workflow using Microsoft Excel, Power Query, and Microsoft Power BI. The project focuses on analyzing a real estate property listings dataset to generate meaningful business insights through data cleaning, transformation, visualization, and interactive dashboard creation.
The project includes data preprocessing in Excel, transformation using Power Query, creation of DAX measures in Power BI, and the development of an interactive dashboard with KPI cards and charts.
